# 上游与修改说明

本版本基于以下公开项目进行适配和修改：

- https://github.com/kuxiaowo/AIpet-Murasame
- https://github.com/LemonQu-GIT/MurasamePet

主要修改方向：

- Apple Silicon macOS 运行环境和启动脚本；
- GPT-SoVITS 本地服务自检、重启和 CPU 运行适配；
- 屏幕与摄像头读取；
- 语音输入、长期记忆和桌面交互；
- macOS 置顶、触控板移动、缩放和托盘菜单。

请勿将角色图片、语音、模型和字体等第三方素材误认为本代码的原创内容。发布这些素材前应单独确认授权。
