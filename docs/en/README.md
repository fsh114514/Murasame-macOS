# Murasame

An Apple Silicon macOS adaptation based on [kuxiaowo/AIpet-Murasame](https://github.com/kuxiaowo/AIpet-Murasame) and [LemonQu-GIT/MurasamePet](https://github.com/LemonQu-GIT/MurasamePet).

This release includes macOS startup support, local GPT-SoVITS integration, cloud chat and vision APIs, screen and camera capture, long-term memory, and desktop interaction improvements.

See the Chinese [README](../../README.md) for the complete release scope, setup instructions, asset notices, and license information.
